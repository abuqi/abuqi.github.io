前言

快速删除docker镜像，容器等。

AviorX安全可靠的企业应用

列出所有的容器 ID

1

docker ps -aq

停止所有的容器

1

docker stop $(docker ps -aq)

删除所有的容器

1

docker rm $(docker ps -aq)

删除所有的镜像

1

docker rmi $(docker images -q)

复制文件

1
2

docker cp container:/opt/file.txt /opt/local/
docker cp /opt/local/file.txt container:/opt/

进入容器 .

1

docker exec -it xxx bin\sh
